Portofolio Pribadi – NURATIA

Ini adalah website portofolio sederhana berisi profil, keahlian, galeri proyek, serta fitur interaktif seperti toggle mode gelap, animasi judul, dan tombol sapaan.

Isi Proyek

index.html — Struktur halaman portofolio
style.css — Tampilan visual, layout, dan responsive design
script.js — Interaksi seperti dark mode, animasi ketik, dan greeting


Fitur

Mode gelap (Dark Mode)
Animasi teks header
Interaksi sapaan pengguna
Layout responsive: Grid & Flexbox
Desain modern dengan warna biru

Tujuan

Proyek ini dibuat sebagai latihan Git, GitHub, HTML, CSS, dan JavaScript dasar.
